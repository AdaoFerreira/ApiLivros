# Api de Livros

## SGDB usado

- Microsoft Sql Server

## Sobre essa api

Nessa api pode adicionar, editar, remover, exibir livros e livro por id. Com
os seguintes items: Título do livro, nome do autor, número de páginas, nome
da editora e o idioma.
